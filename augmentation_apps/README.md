# augmentation_apps

## Vanilla Predictions
A basic application to check the language models' prediction. Supports `roberta`, `bert`, `bart`, `electra`, `xlmroberta`.

### Usage 

`pip install streamlit`

`streamlit run app.py`

### Application Details

Application will `<mask>` the sentence if not `<masked>`
